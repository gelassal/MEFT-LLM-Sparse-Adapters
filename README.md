# Memory-Efficient Fine-Tuning (MEFT)

This repository contains two projects exploring memory-efficient fine-tuning techniques for large language models.

## Repository Structure

```text
MEFT/
├── Project1/
├── Project2/
└── README.md
```

### Project 1

Implementation and evaluation of the original MEFT framework.

### Project 2

Extension of Project 1 that investigates dynamic rank selection and adaptive memory allocation strategies for parameter-efficient fine-tuning.
